# SwanFlip Use Cases

## Student User
- View product prices listed by other UWA students
- View the most sold products in the recent month
- Create a listing for an item to sell
- Edit or delete their own listing
- View seller ratings and verification badges
- Report a listing for admin review
- Suggest a safe meeting location for exchange
- Mark an item as **urgent sale / must go**
- List an item for free
- Save a category and receive notifications for new posts
- Filter products by price
- Change password through a registered email link

## Admin
- View all users in the SwanFlip system
- Review flagged listings
- Approve, warn, or remove listings
- Suspend or remove users for repeated violations
- Access buyer-seller conversations for security purposes

## System
- Display newly created listings to other UWA students
- Notify sellers when their listing is removed
- Notify users when new items are posted in their saved categories
